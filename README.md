# OU-Volatility_Modeling
# Title
Modeling Stock Market Volatility using Ornstein-Uhlenbeck Process

# Overview
- Modeled S&P 500 volatility using stochastic processes
- Used OU process to capture mean reversion
- Estimated parameters via Maximum Likelihood

  # Key Results
  - Volatility is mean-reverting
  - Long-run volatility = 17%
  - Half-life = 1.3 year

  # Methods
  - Rolling Volatilty (40-day window)
  - OU transition density
  - MLE optimization
 
  # Visuals
  - Volatility plot
  - Histogram
  - Simulated Paths
